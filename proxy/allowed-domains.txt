# Sandbox egress allowlist. One pattern per line. # for comments.
#
# Squid `dstdomain` patterns:
#   - `example.com`   matches exactly example.com (no subdomains)
#   - `.example.com`  matches example.com AND every subdomain
#
# Don't list both — Squid 6+ treats redundancy as a fatal config error
# and won't start. Pick the broader form (`.example.com`) when in
# doubt; it's a superset.
#
# Default content below covers the common agent install paths
# (pip, npm, git clone from GitHub). Edit to fit your workload.
# An EMPTY file means "deny everything" — use that as the locked-down
# production default and only allow what your agent actually needs.
